## 5. Known pitfalls

- `package-lock.json` の diff が大きく diff size warning が出るが、初回 npm install のため想定内
- `terminal-sidecar-requirements.md` がリポジトリに存在しない（Issue #2 の参考文書）。後続 Issue 実装時に内容を確認すること
- `onDidWriteTerminalData` は proposed API。本番動作には VS Code を `--enable-proposed-api a2m-m.shirouto-code` で起動する必要がある。Issues #3/#4 実装時に注意
- `contributes.terminal.profiles` は VS Code 1.73+ の API。`registerTerminalProfileProvider` と組み合わせて使う
